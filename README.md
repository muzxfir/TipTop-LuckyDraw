# 🎮 About The Game

**TipTop Lucky Draw** is a modern web-based lucky draw game where users can select a lucky number from **1 to 9** and receive a random prize with exciting visual effects.

Designed for:
- 🎊 Events
- 🏢 Company Activities
- 🎁 Promotions
- 🎉 Fun Games
